TPFINALFINANZAS


Requisitos
- NET SDK 8
- Visual Studio 2022 o VS Code
- SQL Server LocalDB en Windows

Conexion a BD
- appsettings json usa (localdb)\MSSQLLocalDB y crea TPFINALFINANZASDB automaticamente con EnsureCreated

Ejecutar
- Visual Studio 2022  F5
- VS Code  dotnet restore  dotnet build  dotnet watch run

Rutas
- /Usuarios  /Categorias  /Gastos

Flujo
- crear usuario  crear categoria  crear gasto
